Fake News Detection using NLP
This project detects whether a news article is Fake or Real using Natural Language Processing and Machine Learning.

Dataset
Fake and Real News Dataset from Kaggle.

Technologies Used
Python
Pandas
Scikit-learn
TF-IDF Vectorization
Support Vector Machine (SVM)
Workflow
Load dataset
Combine title and text
Clean text data
Convert text to TF-IDF vectors
Train SVM classifier
Predict fake or real news
Accuracy
Model achieves around 98–99% accuracy.

Dataset Link
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset
