# Fake News Detection using NLP

This project detects whether a news article is **Fake or Real** using Natural Language Processing and Machine Learning.

## Dataset

Fake and Real News Dataset from Kaggle.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* TF-IDF Vectorization
* Support Vector Machine (SVM)

## Workflow

1. Load dataset
2. Combine title and text
3. Clean text data
4. Convert text to TF-IDF vectors
5. Train SVM classifier
6. Predict fake or real news

## Accuracy

Model achieves around **96–97% accuracy**.
